# Pinout

| Sensor    | Flipper   |
-------------------------
| 1 - VDD   | 9  - 3V3  |
| 2 - GND   | 18 - GND  |
| 3 - TXD   | 7  - C3   |
| 4 - RXD   | 6  - B2   |
| 5 - SCL   | 16 - C0   |
| 6 - SDA   | 15 - C1   |
